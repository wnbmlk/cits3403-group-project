# User Stories
1. As a user, I want to sign up for an account so I can access personalised features.
1. As a user, I want to log in so that my progress is saved and I can continue tracking my movies later
1. As a user, I want to add a movie that I have watched so I can track my viewing history.
1. As a user, I want to rate a movie so that I can record my opinion.
1. As a user, I want to write a review when adding a movie so I can express my thoughts on the movie.
1. As a user, I want to view a timeline of movies that I’ve watched so I can see my activity over time.
1. As a user, I want to see graph visualisations of my movie timeline so I can analyse my watching habit in the past.
1. As a user, I want to search for movies so I can quickly find and add them to my watched list.
1. As a user, I want to be able to interact and connect with other users.
1. As a user, I want to view my connections’ timelines so I can see what they have been watching.
1. As a user, I want to be able to log out so that I can securely exit my account.
